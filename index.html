3.1 开关代数

3.1.1 基本公理

\[ （A1）X = 0 \quad if X \neq 1, （A1’） X = 1 \quad if X \neq 0 \\\\ （A2）If X = 0,then X’ = 1（A2’）If X = 1 , then X’ =0 \\\\ （A3） 0·0 = 0 （A3’） 1+1 = 1 \\\\ （A4） 1·1 = 1 （A4’） 0+0 = 0 \\\\ （A5） 0·1 = 1·0 = 0 （A5’） 1+0 = 0+1 = 1 \]

3.1.2 单变量开关代数定理

\[ Identities (自等律)：\\(T1) X + 0 = X \quad(T1’) X · 1 = X \\ Null Elements(0-1律)：\\(T2) X + 1 = 1 \quad (T2’) X · 0 = 0 \\ Idempotency (同一律):\\ (T3) X + X = X \quad(T3’) X · X = X\\ Involution (还原律)：\\(T4) ( X’)’ = X\\ Complements (互补律): \\(T5) X + X’ = 1 \quad (T5’) X · X’ =0 \]

3.1.3 多变量开关代数定理

\[ Commutativity (交换律)\\ (T6)X + Y = Y + X \quad (T6’) X · Y= Y ·X \\ Associativity (结合律)\\ (T7) X·(Y·Z) = (X·Y)·Z \quad (T7’) X+(Y+Z) = (X+Y)+Z\\ Distributivity (分配律)\\ (T8) X·(Y+Z) = X·Y+X·Z \quad (T8’) X+Y·Z = (X+Y)·(X+Z)\\ Covering(吸收律)\\ (T9) X + X·Y = X \quad (T9’) X·(X+Y) = X\\ Combining(合并律)\\ (T10) X·Y + X·Y’ = X \quad (T10’) (X+Y)·(X+Y’) = X\\ Consensus（添加律（一致性定理））\\(T11) X·Y + X’·Z + Y·Z = X·Y + X'·Z\\ (T11’) (X+Y)·(X’+Z)·(Y+Z) = (X+Y)·(X’+Z) \] 代入定理： 在含有变量 X 的逻辑等式中，如果将式中所有出现 X 的地方都用另一个函数 F 来代替，则等式仍然成立。

3.1.4 n变量定理

\[ Generalized idempotency(广义同一律)\\ (T12) X + X + … + X = X \quad(T12’) X · X · … · X = X\\ DeMorgan’s Theorems(德.摩根定理)\\ (T13) (X1·X2·……·X n)’=X1’+ X2’ +……+X n’\\ (T13’) (X1+ X2+ ……+X n)’= X1’ · X2’ ·……· X n’\\ Generalized DeMorgan’s Theorems (广义德.摩根定理)\\ (T14)[F(X1 ,X2 …,X n,+, ·)]’=F(X1’, X2’,…,X n’, ·,+) \] 反演定理： \[ [F(X_1, X_2, \dots, X_n, +, \cdot, 0, 1)]' = F(X_1', X_2', \dots, X_n', \cdot, +, 1, 0) \]

3.1.5 对偶定理

对开关代数的任何定理和恒等式，若交换所有的0和1以及“+”和“· ”，结果仍然正确 \[ F^D(X1 , X2 , … , Xn , + , · , ’ ) = F(X1 , X2 , … ,Xn , · , + , ’ ) \] 香农展开定理： \[ F(X_1, X_2, \dots, X_n) = X\,F(1, X_2, \dots, X_n) + X'\,F(0, X_2, \dots, X_n) \]

3.1.6 逻辑函数的标准表示方式

最小项： n个因子的标准乘积项，每个变量都以原变量或反变量的形式且只出现一次。 - 任意两个不同最小项的乘积为0 - 全体最小项之和为1

最小项公式： \[ m_i = \prod_{k=1}^{n} x_k, \quad x_k = \begin{cases} X_k', & i_k=0 \\ X_k, & i_k=1 \end{cases} \] 对三变量 \(A, B, C\)，若 \(F=1\) 的最小项编号为 \(i_1, i_2, \dots, i_k\)，则 \[ F(A, B, C) = \sum_{X,Y,Z} m(i_1, i_2, \dots, i_k) \] 其中 \(\sum_{X,Y,Z} m(i_1, i_2, \dots, i_k)\) 表示函数值为 1 的那些最小项之和。

最大项： 是具有n个因子的标准求和项，每个变量都以原变量或反变量的形式且只出现一次。 - 任意两个最大项的和为1 - 全体最大项之积为0

最大项公式： \[ M_i = \sum_{k=1}^{n} y_k, \quad y_k = \begin{cases} X_k, & i_k=0 \\ X_k', & i_k=1 \end{cases} \] 对三变量 \(A, B, C\)，若 \(F=0\) 的最大项编号为 \(j_1, j_2, \dots, j_k\)，则 \[ F(A, B, C) = \prod_{X,Y,Z} M(j_1, j_2, \dots, j_k) \] 其中 \(\prod_{X,Y,Z} M(j_1, j_2, \dots, j_k)\) 表示函数值为 0 的那些最大项之积。

求和项 A B C 最大项编号
\((A+B+C)\) 0 0 0 \(M_0\)
\((A+B+C')\) 0 0 1 \(M_1\)
\((A+B'+C)\) 0 1 0 \(M_2\)
\((A+B'+C')\) 0 1 1 \(M_3\)

\((A'+B'+C)\) 1 1 0 \(M_6\)
\((A'+B'+C')\) 1 1 1 \(M_7\)
  • 编号相同的最小项和最大项互为反函数 \(m_i=(M_i)’\)
  • 某逻辑函数 F，若用 P 项最小项之和表示，则其反函数 F’ 可用 P 项最大项之积表示，两者标号完全一致。
  • 一个 n 变量函数，既可用最小项之和表示，也可用最大项之积表示，两者下标互补。
  • 一个 n 变量函数的最小项 mi,其对偶(Duality)为： \(( m _i )^D = M_{(2 n -1) - i}\)

3.3.4 Karnaugh Maps（卡诺图）

  1. 逻辑函数F的最小和是一个积之和的表达式，它比其它的积之和表达式乘积项个数更少，而且每个乘积项数中变量的个数也是最小的，其中0为反变量，1为正变量。
  2. 对于逻辑函数 P(X1,…,Xn) 和 F(X1,…,Xn) ，若对任何使P=1的输入组合，也能使F为1，则称P隐含F，或者F包含P。 P为F的蕴含项（implicant），在卡诺图上对应的是只包含1的一个矩形圈。
  3. 主蕴含项（prime implicant） ：扩展到最大的蕴含项。最小和是主蕴含项的集合。
  4. 逻辑函数中的奇异一单元（distinguished 1-cell）是在卡诺图中仅被单一主蕴含项覆盖的输入组合。（包括不同的化简方式）。
  5. 逻辑函数F中的质主蕴含项是覆盖一个或多个奇异一单元的主蕴含项。
  • 卡诺图形式 卡诺图的化简方式
  1. 填写卡诺图
  2. 圈组：找出可以合并的最小项 先找奇异“1”单元，圈质主蕴涵项，再圈其它项 保证每个圈的范围尽可能大、圈数尽可能少 方格可重复使用，但不要重叠圈组,至少有一个1未被圈过
  3. 读图：写出化简后的各项 消掉既能为0也能为1的变量 保留始终为0或始终为1的变量

无关输入组合（ don’t care Input Combinations）： The output doesn’t matter for certain input combination (maybe never occur in normal condition). These are called don’t care terms. Use symbol “d”、“×”、“ Φ” to represent the output value. In minimization, don’t care term could be used as “1” or “0” if necessary.

最小积： 在卡诺图中通过圈0找出最小和之积的形式，其中0为原变量，1为反变量。 在卡诺图中圈1的代价和圈0的代价一般不同。 the minimal NAND-NAND expresstion: 最小和取非后再取非。 the minimal NOR-NOR expresstion: 最小积取非后再取非。 AOI expresstion： 最小和反函数后取非。

3.4 Timing Hazards(定时冒险)

A hazard is said to exist when a circuit has the possibility of producing such a glitch（尖峰）.

3.4.1 静态冒险

static-1 hazard：稳定状态为1，常出现于“与——或”两级电路中。 static-0 hazard：稳定状态为0，常出现于“或——与”两级电路中。

3.4.2 Finding Static Hazards Using Maps

若卡诺图中，圈与圈之间有相切（tangent）现象，则可能出现静态冒险。 消除冒险的方法：引入额外项乘积项覆盖冒险的输入对。

3.4.3 Dynamic hazard（动态冒险）

Happened in the circuits which have more than two levels ! Different signal pass may have different time delay ! When input changed once, output may be changed several times !

竞争： 门电路两个输入信号同时向相反的逻辑电平跳变。 竞争－冒险：由于竞争而在电路输出端可能产生尖峰脉冲。 消除竞争－冒险 现象的办法： 1. 引入滤波电容：尖峰脉冲一般都很窄，输出端并接一个很小的滤波电容，足以将其幅度削弱到门电路的阈值电压以下。（缺点：会增加输出电压波形的上升和下降时间） 2. 引入选通脉冲。 3. 修改逻辑设计。

第一章 半导体基础及二极管应用电路

1.1 半导体基础知识

1.1.1本征半导体

  1. 共价键结构 价电子被束缚在共价键的状态，价电子可能占有的能级位于较低的有效能带中，该能带称为能带。 自由电子处于自由状态，其能量状态较高，自由电子可能占有的能级也是量子化的，位于较高的能带内，该能带称为导带。 自由电子可能占有的最低能级要比价电子可能占有的最高能级高出\(E_g\)，\(E_g\)称为禁带宽度，不可能被电子占据。
  2. 本征激发与两种载流子 共价键中留下空位时，在外加电场的作用下，邻近共价键中的价电子就可能来补充这个空位，同时在邻近共价键中产生新的空穴，这样就相当于空穴在单晶体中随机运动。
  3. 本征载流子浓度 电子与空穴的产生与复合最终达到一种热平衡状态，使本征半导体中的本征载流子浓度处于某一热平衡的统计值。 \[ n_i = p_i = \sqrt{N_c N_v}\,e^{-\frac{E_g}{2kT}} \]

1.1.2 杂质半导体

  1. N型半导体 掺入五价元素产生的多余电子所占有的能级较高，很靠近导带底部，称为施主能级。
  2. P型半导体 掺入三价元素形成的受主能级一般很靠近价带顶部。

1.1.3漂移电流与扩散电流

  1. 漂移电流 载流子在电场作用下的定向运动称为漂移运动，由漂移运动产生的电流叫漂移电流。
  2. 扩散电流 在浓度差的作用下产生了非平衡载流子的扩散运动。

1.2 PN结

1.2.1 PN结的形成

  1. PN结的形成 界面两侧附近将形成一个由正离子和负离子构成的空间电荷区，电荷区存在着内建电场。内建电场的形成阻止两区多子的扩散，有利于形成两区少子的漂移运动。当扩散电流与偏移电流相等时，达到动态平衡，PN结形成。
  2. PN结的特点 1 空间电荷区的宽度决定于杂质浓度。杂质浓度越高，空间电荷区越薄，空间电荷区向杂质浓度低的一侧延伸。 2 空间电荷区是非中性区 3 空间电荷区也称为势垒区或阻挡层。 4 结内载流子远小于结外载流子数，PN结又称耗尽层。

1.2.2 PN结的单向导电特性

偏置（bias）泛指在半导体器件上所加的直流电压和电流。 1. 正向偏置的PN结 2. 反向偏置的PN结

1.3.6PN结电容效应及其应用

第二章 晶体三极管基础

2.1 双极型晶体三极管

2.1.1 BJT的工作原理

  1. BJT结构 BJT是由两个N区夹一个P区或由两个P区夹一个N区形成的具有两个PN结的结构。中间所夹的异型杂质半导体区称为基区（base）。另外两个杂质半导体区分别称为发射区（emitter）和集电区（collector）。发射区与基区之间的PN结称为发射结（BE结），集电区与基区之间的PN结称为集电结(CB结)。
  2. BJT的三种基本组态 共射，共集，共基
  3. BJT偏置方式及电流分配关系 1 发射结正偏，集电结反偏，工作在放大状态。 2 发射结正偏，集电结正偏，工作在饱和状态。 3 发射结反偏，集电结反偏，工作在截止状态。 4 发射结反偏，集电结正偏，工作在反向运行状态。由于双极性晶体管的发射结和集电结是不对称的，应避免反向运行的工作状态。 (1)BJT的放大偏置方式 NPN管：\(U_C > U_B> U_E\) PNP管：\(U_C < U_B< U_E\) (2)放大偏置时BJT内部载流子的传输过程 1 发射区多子向基区扩散（又称注入） 发射区向基区扩散的自由电子电流\(i_{En}\)远大于基区向发射区扩散的空穴电流\(i_{Ep}\)，即\(i_E\approx i_{En}\)。 2 基区非平衡少子向集电结方向边扩散边复合 非平衡自由电子在基区的扩散过程中，由于基区的杂质浓度低，且基区做的很薄，其中只有一小部分少子在扩散过程中与多数载流子复合，形成基区复合电流\(i_{B1}\)。基区复合电流\(i_{B1}\)是基极电力的主要成分，表示了从基极引线进入基区的空穴电流。 3 集电区收集基区非平衡少子 大部分扩散到集电结边缘的少子在集电结反向偏置电场作用下被迅速收集进入集电区，形成集电极电流的主要成分\(i_{Cn1}\)。 除此之外，在反偏集电结电压所产生电场的作用下，基区的少子也会越过集电结漂移到达集电区形成漂移电流\(i_{Cn2}\);集电区中的少子越过集电结漂移到基区形成漂移电流\(i_{Cp}\)。\(i_{Cp}+i_{Cn2}=I_{CBO}\)就是集电结的反向饱和电流。 (3)放大偏置时的电流关系

  • 基本电流关系 \[ i_E = i_B + i_C \]

  • 放大区的近似关系 \[ i_C \approx \alpha i_E + I_{CBO} \] \[ i_B \approx (1-\alpha)i_E - I_{CBO} \]

  • 用电流放大系数表示 \[ i_C \approx \beta i_B + (1+\beta)I_{CBO} \] 其中，\(\alpha\)为共基电流放大系数，\(\beta\)为共射电流放大系数，且有 \[ \beta = \frac{\alpha}{1-\alpha} \]

2.1.2 BJT的静态特性曲线

  1. 共射输入特性曲线 共射输入特性曲线是指BJT在共射组态下。输入端口的直流电流与电压的伏安特性曲线族。通常它以输出端口的电压\(u_{CE}\)为参变量，反应输入端口基极电流\(i_B\)随发射结电压\(u_{BE}\)变化的特性曲线。
  2. 共射输出特性曲线 共射输出特性曲线是以输入端口电流\(i_B\)为参变量，反应输出端口集电极电流\(i_C\)随输出端口集-射电压\(u_{CE}\)变化的特性曲线。 (1)放大区 在该区域发射区正偏，集电区反偏，参变量\(i_B>0\)。 (2)饱和区 发射结和集电结均处于正偏导通状态。 在饱和区各条曲线几乎重合在一起，表明比例关系不再成立。在饱和区\(u_{CE}\)的值被称为饱和压降，记为\(U_{CES}\)。 (3)截止区 发射结和集电结均处于反偏状态，集电极电流为反向饱和电流（\(i_C=I_{CBO}\)）。工程上往往认为参变量\(I_B\)小到等于零时，BJT就截止了，此时\(i_C=I_{CEO}\)。 (4)击穿区 当\(u_{CE}\)增大到一定值时，集电结会发生反向击穿，\(i_C\)急剧增大。
  3. 温度对BJT特性曲线的影响 当温度增加时，共射输入特性曲线会左移。共射输出特性曲线会随温升而上移。
