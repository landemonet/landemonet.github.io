数字电路笔记

第三章、开关代数和组合逻辑

3.1 开关代数

3.1.1 基本公理

$$
（A1）X = 0 \quad if X \neq 1, （A1’） X = 1 \quad if X \neq 0 \
（A2）If X = 0,then X’ = 1（A2’）If X = 1 , then X’ =0 \
（A3） 0·0 = 0 （A3’） 1+1 = 1 \
（A4） 1·1 = 1 （A4’） 0+0 = 0 \
（A5） 0·1 = 1·0 = 0 （A5’） 1+0 = 0+1 = 1
$$

3.1.2 单变量开关代数定理

$$
Identities (自等律)：\ (T1) X + 0 = X \quad(T1’) X · 1 = X \
Null Elements(0-1律)：\ (T2) X + 1 = 1 \quad (T2’) X · 0 = 0 \
Idempotency (同一律):\ (T3) X + X = X \quad(T3’) X · X = X\
Involution (还原律)：\ (T4) ( X’)’ = X\
Complements (互补律): \ (T5) X + X’ = 1 \quad (T5’) X · X’ =0
$$

3.1.3 多变量开关代数定理

$$
Commutativity (交换律)\ (T6)X + Y = Y + X \quad (T6’) X · Y= Y ·X \
Associativity (结合律)\ (T7) X·(Y·Z) = (X·Y)·Z \quad (T7’) X+(Y+Z) = (X+Y)+Z\
Distributivity (分配律)\ (T8) X·(Y+Z) = X·Y+X·Z \quad (T8’) X+Y·Z = (X+Y)·(X+Z)\
Covering(吸收律)\ (T9) X + X·Y = X \quad (T9’) X·(X+Y) = X\
Combining(合并律)\ (T10) X·Y + X·Y’ = X \quad (T10’) (X+Y)·(X+Y’) = X\
Consensus（添加律（一致性定理））\(T11) X·Y + X’·Z + Y·Z = X·Y + X’·Z\ (T11’) (X+Y)·(X’+Z)·(Y+Z) = (X+Y)·(X’+Z)
$$
代入定理：
在含有变量 X 的逻辑等式中，如果将式中所有出现 X 的地方都用另一个函数 F 来代替，则等式仍然成立。

3.1.4 n变量定理

$$
Generalized idempotency(广义同一律)\
(T12) X + X + … + X = X \quad(T12’) X · X · … · X = X\
DeMorgan’s Theorems(德.摩根定理)\
(T13) (X1·X2·……·X n)’=X1’+ X2’ +……+X n’\
(T13’) (X1+ X2+ ……+X n)’= X1’ · X2’ ·……· X n’\
Generalized DeMorgan’s Theorems (广义德.摩根定理)\
(T14)[F(X1 ,X2 …,X n,+, ·)]’=F(X1’, X2’,…,X n’, ·,+)
$$
反演定理：
$$
[F(X_1, X_2, \dots, X_n, +, \cdot, 0, 1)]’ = F(X_1’, X_2’, \dots, X_n’, \cdot, +, 1, 0)
$$

3.1.5 对偶定理

对开关代数的任何定理和恒等式，若交换所有的0和1以及“+”和“· ”，结果仍然正确
$$
F^D(X1 , X2 , … , Xn , + , · , ’ ) = F(X1 , X2 , … ,Xn , · , + , ’ )
$$
香农展开定理：
$$
F(X_1, X_2, \dots, X_n) = X,F(1, X_2, \dots, X_n) + X’,F(0, X_2, \dots, X_n)
$$

3.1.6 逻辑函数的标准表示方式

Truth table 真值表
product term 乘积项 sum term 求和项
A sum-of-products expression “积之和”表达式
product-of-sums expression “和之积”表达式
n-variable minterm n 变量最小项
n-variable maxterm n 变量最大项
normal terms 标准项
canonical sum 标准和
canonical product 标准积

最小项： n个因子的标准乘积项，每个变量都以原变量或反变量的形式且只出现一次。

  • 任意两个不同最小项的乘积为0
  • 全体最小项之和为1

最小项公式：
$$
m_i = \prod_{k=1}^{n} x_k, \quad x_k = \begin{cases}
X_k’, & i_k=0 \
X_k, & i_k=1
\end{cases}
$$
对三变量 $A, B, C$，若 $F=1$ 的最小项编号为 $i_1, i_2, \dots, i_k$，则
$$
F(A, B, C) = \sum_{X,Y,Z} m(i_1, i_2, \dots, i_k)
$$
其中 $\sum_{X,Y,Z} m(i_1, i_2, \dots, i_k)$ 表示函数值为 1 的那些最小项之和。

乘积项 A B C 最小项编号
$A’*B’*C’$ 0 0 0 $m_0$
$A’*B’*C$ 0 0 1 $m_1$
$A’BC’$ 0 1 0 $m_2$
$A’BC$ 0 1 1 $m_3$
$A*B’*C’$ 1 0 0 $m_4$
$A*B’*C$ 1 0 1 $m_5$
$ABC’$ 1 1 0 $m_6$
$ABC$ 1 1 1 $m_7$

最大项： 是具有n个因子的标准求和项，每个变量都以原变量或反变量的形式且只出现一次。

  • 任意两个最大项的和为1
  • 全体最大项之积为0

最大项公式：
$$
M_i = \sum_{k=1}^{n} y_k, \quad y_k = \begin{cases}
X_k, & i_k=0 \
X_k’, & i_k=1
\end{cases}
$$
对三变量 $A, B, C$，若 $F=0$ 的最大项编号为 $j_1, j_2, \dots, j_k$，则
$$
F(A, B, C) = \prod_{X,Y,Z} M(j_1, j_2, \dots, j_k)
$$
其中 $\prod_{X,Y,Z} M(j_1, j_2, \dots, j_k)$ 表示函数值为 0 的那些最大项之积。

求和项 A B C 最大项编号
$(A+B+C)$ 0 0 0 $M_0$
$(A+B+C’)$ 0 0 1 $M_1$
$(A+B’+C)$ 0 1 0 $M_2$
$(A+B’+C’)$ 0 1 1 $M_3$
$(A’+B+C)$ 1 0 0 $M_4$
$(A’+B+C’)$ 1 0 1 $M_5$
$(A’+B’+C)$ 1 1 0 $M_6$
$(A’+B’+C’)$ 1 1 1 $M_7$
  • 编号相同的最小项和最大项互为反函数 $m_i=(M_i)’$
  • 某逻辑函数 F，若用 P 项最小项之和表示，则其反函数 F’ 可用 P 项最大项之积表示，两者标号完全一致。
  • 一个 n 变量函数，既可用最小项之和表示，也可用最大项之积表示，两者下标互补。
  • 一个 n 变量函数的最小项 mi,其对偶(Duality)为：
    $( m i )^D = M{(2 n -1) - i}$
